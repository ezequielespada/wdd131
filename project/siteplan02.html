Project Site Plan

Site Name

Ladousse Medical Workwear

El nombre refleja el enfoque en indumentaria médica y transmite un toque de elegancia y distinción.

Site Purpose

El sitio web busca ofrecer una plataforma de compra en línea de ropa médica profesional, como uniformes, chaquetas y accesorios para trabajadores de la salud.

Scenarios

  • ¿Qué tipos de uniformes están disponibles para enfermeros?
  • ¿Dónde puedo encontrar información de contacto para hacer una compra mayorista?

Color Schema

  • Azul (#002B7F) - Encabezados y botones
  • Rojo (#E4002B) - Elementos destacados y acentos
  • Blanco (#FFFFFF) - Fondo de la página

Typography

  • Roboto - Encabezados y títulos
  • Open Sans - Cuerpo del texto
